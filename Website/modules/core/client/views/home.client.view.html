Rajasthan Governement Health Empowerment and Research Project

Welcome to Health Empowerment project by govenrment of rajasthan, built with aim for empowerment of every healthcare service consumer to reach them in need and be with 24*7.

Features

  • Built with support of E-Mitra.
  • Designed with MEAN.JS, adavance and fast web technology.
  • speech recognition and chatbots for easier use.
  • Web portal with mobile app, kiosk interface and avtive workers on ground.
  • Artificial intelligence for learning from user and hence automating processes and decreasing mannual efforts, also a big game changer for using data for medical research.
